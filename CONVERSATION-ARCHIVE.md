## Why Separate?

The conversation archive represents unique intellectual property and meta-content that is separate from the Claude Role Framework project itself:

- **Project Repository:** Clean, professional, ready for Anthropic review
- **Conversation Archive:** Private, protected, used for digital twin development

## Access

```bash
# View archive
cd ~/Projects/ai-conversation-archive/claude-role-framework-phase1/

# Quick start
cat ~/Projects/ai-conversation-archive/claude-role-framework-phase1/QUICK-START.md

# Search patterns
jq '.thinking_patterns.patterns[].name' \
  ~/Projects/ai-conversation-archive/claude-role-framework-phase1/indexes/thinking-patterns.json
```

## Universal Tool

The conversation analysis prompt is available for use in any AI conversation:

```bash
cat ~/Projects/ai-conversation-archive/CONVERSATION-ANALYSIS-PROMPT.md
```

Copy/paste into any AI chat to extract thinking patterns and insights.
